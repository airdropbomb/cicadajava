# cicada auto task

## Installation

1. Clone the repository:
```bash
git clone https://github.com/airdropbomb/cicada.git && cd cicada
```

2. Install dependencies:
```bash
npm install
```

3. input bearer & cookie in `accounts.json` file in the root directory:
```bash
nano accounts.json
```

4. ## Usage

1. Start the bot:
```bash
node index.js
```
